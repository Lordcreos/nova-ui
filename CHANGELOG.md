# Changelog

All notable changes to `@nova-ui/components` will be documented here.

## [0.1.0] — 2026-05-11

### Added

- Initial release with 40+ production-ready components
- **UI components**: Accordion, Alert, Avatar, Badge, Breadcrumb, Button, Calendar, Card, Checkbox, Dropdown Menu, Empty State, Helper Text, Icon, Input, Label, Modal, Page Tabs, Pagination, Popover, Progress, Radio Button, Row Card, Scroll Area, Segmented Control, Separator, Select, Skeleton, Spinner, Switch, Table, Textarea, Toast, Toggle, Tooltip
- **Form components**: TextField, TextareaField, FileDropZone, FileItem, GroupTextField, SearchableSelect
- **Blocks**: Logs, TablePagination
- **Typography**: Heading, Body
- **Hooks**: `useToast`
- Design token system with 7 CSS files (colors, typography, shadows, sizing, responsive, animations)
- Full TypeScript support with strict mode
- CVA (class-variance-authority) variant system throughout
- Radix UI accessibility primitives
- Storybook documentation for all components
